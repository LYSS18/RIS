# 轨迹预测误差对运动员健康指标影响的误差分析


## 实验文件

### `health_risk_analysis.py`

**轨迹预测误差分析实验**

基于Error Analysis部分的完整数学模型，实现误差传播分析。

**核心功能**：

- **功率变化计算**: 计算预测误差导致的功率变化
- **角度干扰分析** : 分析角度预测误差对干扰的影响
- **SINR降级计算** : 计算轨迹预测误差对SINR的影响
- **数据包错误率** : 基于SINR计算传输可靠性
- **监测延迟模型** : 计算生理监测延迟
- **生命安全风险** : 量化对运动员安全的影响

## 论文数学模型

### 误差传播链条

```
轨迹预测误差 → SINR降级 → 数据包错误率 → 生理监测延迟 → 生命安全风险
```

### 关键公式

**功率变化** :
```
ΔP_total = P_total' - P_total
```

**SINR降级** :
```
ΔSINR = P_sig'/(P_int'+P_noise) - P_sig/(P_int+P_noise)
```

**心脏风险** :
```
P_cardiac,miss = 1 - exp(-λ_cardiac × ΔT_cardiac)
```

**综合风险** :
```
R_safety = w_cardiac × P_cardiac + w_thermal × P_thermal
```
